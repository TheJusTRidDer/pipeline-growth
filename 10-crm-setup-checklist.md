# CRM Setup Checklist — HubSpot Free

## Day 1: Pipeline + Properties

- [ ] Sign up at hubspot.com (free)
- [ ] Settings → CRM → Pipelines → Deals → Create "Growth Sales" pipeline
- [ ] Create 10 stages: Prospect → Contacted → Responded → Discovery Call → Audit in Progress → Proposal Sent → Negotiation → Closed Won → Closed Lost → Nurture
- [ ] Create custom properties:
  - Client Tier (dropdown: Audit / Growth Engine / Growth Partner)
  - Lead Source (dropdown: Referral / LinkedIn / Cold Email / Partner / Website / Other)
  - Priority Score (number 0-100)
  - Pipeline Value (currency)
  - Pain Point (multi-checkbox)
  - Budget Signal (dropdown: High / Medium / Low / Unknown)

## Day 2: Import Clients + Templates

- [ ] Create CSV with existing clients → Import into Contacts
- [ ] Settings → Integrations → Email → Connect your inbox
- [ ] Enable "Log emails automatically"
- [ ] Create 5 email templates (from 03-cold-email-sequence.md):
  1. Direct Ask
  2. Insight Hook (Referral Math)
  3. Case Study
  4. Growth Audit offer
  5. Breakup

## Day 3: Lists + Dashboard

- [ ] Create 6 contact lists: Active Pipeline, Warm Leads, Needs Follow-up, Referral Sources, Nurture, Top 20 Prospects
- [ ] Create dashboard with tiles: Pipeline by Stage, Monthly New Deals, Lead Source Breakdown, Close Rate, Monthly Revenue, Upcoming Tasks
- [ ] Set up meeting booking link (CRM → Meetings → Create)

## Daily Workflow (15 min/day)

- [ ] 08:00 — Check pipeline value + deals stuck >7 days
- [ ] 08:05 — Review tasks due today
- [ ] 08:08 — Move deals forward in Active Pipeline
- [ ] 08:12 — Add new prospects from LinkedIn/Apollo
- [ ] 08:15 — Log yesterday's calls/emails

## Weekly Review (30 min/Friday)

- [ ] New prospects added (>50) — Pipeline value ($50k+)
- [ ] Emails sent (200+) — Deals closed (1/2 weeks)
- [ ] Replies (5+) — Discovery calls booked (2+)
