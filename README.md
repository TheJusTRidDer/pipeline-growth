# Pipeline Growth

A complete outbound sales system for professional services firms (5-50 employees).

## What's Inside

| File | What It Is |
|---|---|
| `index.html` | Landing page website (deployed via GitHub Pages) |
| `01-pricing-page.md` | Pricing strategy & tiers |
| `02-referral-email.md` | Referral request email template |
| `03-cold-email-sequence.md` | 7-email cold sequence |
| `04-linkedin-profile.md` | LinkedIn profile optimization |
| `05-growth-audit-template.md` | Growth audit deliverable template |
| `06-lead-list-construction.md` | ICP filters + prospect sourcing |
| `07-crm-pipeline-setup.md` | HubSpot pipeline setup guide |
| `08-sales-call-scripts.md` | 6 sales call scripts |
| `09-personalized-emails-top10.md` | Personalized outreach samples |
| `10-crm-setup-checklist.md` | CRM setup step-by-step checklist |
| `11-email-infra-setup.md` | Email sending infrastructure guide |
| `lead-list-template.csv` | Lead list import template |

## Getting Started

1. Follow `10-crm-setup-checklist.md` to configure HubSpot
2. Follow `06-lead-list-construction.md` to build your first 50 prospects
3. Follow `11-email-infra-setup.md` to set up sending
4. Use `09-personalized-emails-top10.md` to craft first outreach
5. Use `08-sales-call-scripts.md` to handle discovery calls
